# SlowTechno FUKUI

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

An open data project featuring AI-generated slow techno music with a Fukui, Japan theme. The tracks were created using [Suno](https://suno.com/).

This repository provides the generated music as open data, along with a web-based player and a script to download and archive Suno playlists.

## Listen

- **[Interactive Web Player (GitHub Pages)](https://code4fukui.github.io/music-slowtechno-fukui/)**
- **[Original Playlist on Suno](https://suno.com/playlist/1f0f31ae-87b3-438a-ac53-240320b1477e)**

The web player displays the tracklist, album art, lyrics, and the prompts used for generation.

## About the Music

The tracks are inspired by the culture of Fukui, Japan. For example, the song "ソースカツドンの街" (Sauce Katsudon Town) was generated from the following prompt:

> フクイといえばソースカツドン,slow techno

Suno's model generated a detailed style description for the tracks:
> Slow techno with a restrained four-on-the-floor pulse, dry kick and soft clap locked to a patient groove; verse stays sparse with murmured Japanese lead, short bass pulses, and filtered bell ticks, pre-chorus opens with rising noise and echo tails, chorus lands wider with layered unison doubles and chant-like backing repeats.

## Data

This repository contains:
- `audio/`: MP3 audio files for each track.
- `image/` & `image_large/`: Cover art for each track.
- `playlist.json`: The complete playlist data, including lyrics and generation metadata, with local file paths.
- `playlist_org.json`: The original playlist data from Suno with remote URLs.

## Usage: Download a Suno Playlist

You can use the provided Deno script to download any public Suno playlist and its assets (audio, images) locally.

```sh
deno run -A https://code4fukui.github.io/music-opendata-fukui/download.js https://suno.com/playlist/YOUR_PLAYLIST_URL
```
*Example using this repository's playlist:*
```sh
deno run -A https://code4fukui.github.io/music-opendata-fukui/download.js https://suno.com/playlist/1f0f31ae-87b3-438a-ac53-240320b1477e
```
